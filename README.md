# vrinda-store-sales-analysis
End-to-End Excel Data Analytics Project including Data Cleaning, Analysis, Dashboard Design and Business Insights.
## Project Overview

This project analyzes retail sales data for Vrinda Store using Microsoft Excel. The objective was to transform raw sales data into meaningful business insights through data cleaning, KPI reporting, and dashboard visualization.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## Dataset

The repository contains:

* Raw sales data
* Cleaned sales data
* Interactive Excel dashboard
* Business insights

## Data Cleaning Steps

* Removed duplicate records
* Standardized date formats
* Checked for missing values
* Corrected inconsistent entries
* Prepared data for analysis

## Dashboard Features

### KPIs

* Total Sales
* Average Age
* Total Orders
* Average Order Value

### Visualizations

* Monthly Sales Trend
* Sales by Gender
* Sales by State
* Sales by Category
* Sales by Channel

### Interactive Filters

* Month
* Category
* Channel
* Gender

## Key Insights

* Women contributed approximately 64% of total sales.
* March recorded the highest sales performance.
* Amazon was the top-performing sales channel.
* Set category generated the highest revenue.
